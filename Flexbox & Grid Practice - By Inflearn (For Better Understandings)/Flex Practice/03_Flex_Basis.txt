유연한 박스의 기본 영역
Flex item의 기본 크기를 결정한다
width height와 같으면서도 약간 차이가 있다

flex-direction이 row일때는 너비, column일때는 높이

default : auto
숫자값들이 들어갈 수 있음

.item {
	flex-basis: 100px;
}
원래의 width가 100px이 안되는 AAA와 CCC는 100px로 늘어났고, 원래 100px이 넘는 BBB는 그대로 유지된다.
자기 값을 넘으면 그대로 유지, 아니면 100px로 늘어남.

.item {
	width: 100px;
}

width를 설정하면 원래 100px을 넘는 BBB도 100px로 맞춰준다.
https://studiomeal.com/archives/197 참고

Flex Grow - 아이템이 flex-basis 값보다 커질 수 있는지 결정하는 속성이다
숫자값이 들어가는데 0보다 크면 일단 유연한 박스로 변하고 원래의 크기보다 커지면서 빈 공간을 메우게 된다.
flex-basis를 제외한 전체 여백 부분을 flex-grow에 배정된 숫자로 나눠서 배정한다

flex-shrink
flex-grow와 set로 이루어짐. flex-basis 값보다 작아질 수 있는가
기본값 1이라 따로 세팅하지 않아도 flex-basis보다 작아질 수 있다. grow와 같이 비율을 이야기한다.

flex-basis flex-grow flex-shrink -> 하나로 따라다닌다.
